## 一、图幅与图框（标准模板，以后固定使用）

1. **图幅**：A4（297×210），单页出图，一个零件一张图。
2. **边框**：A4 外框（0,0)-(297,210) 保持不变，**加内框（向内偏移 3mm：(3,3)-(294,207)）**；**标题栏右边、底边与内框相连**（右 x294、底 y3），标题栏、表格其余边线自带。
3. **投影法**：**第一角投影法**，投影符号（梯形在左、同心圆在右）与"一角法"字样同置于标题栏符号格下部（第二行区域内），字样在图形正下方并对齐符号组中心线。
4. **图框内容**：
   - 公司名称栏：**老吴ai学习有限公司**（纯文字，无图标/Logo），位于标题栏顶部通栏（高 14mm，字号 5.0）。
   - 左下角固定 3 条技术要求：
     1. 零件表面不可刮花、碰伤；
     2. 去除所有毛刺、锐边；
     3. 未注倒角 C0.5；
   - 右上角注：其余 Ra3.2。
   - 左下角：未注公差分段表（±0.05 ~ ±0.50 分段）。
   - 右下角标题栏（与内框相连：右 x294、底 y3），栏目如下：
     - 第一行（高 14mm）：**零件名称**、**零件图号**（图号与图号值之间无竖分隔线；
       **图号值在图号栏值区内居中（MiddleCenter），长图号自动缩小字号适应栏宽，
       图号文字不得与栏边框干涉**——V1.8）
     - 第二行（高 25mm，两行式）：
       - **比例**（下：单位）｜比例值（下：mm）
       - **材料**（下：表面处理）｜材料牌号（下：表面处理方式，铝件默认"喷砂本色阳极"）
       - **数量**（下：图样）｜数量值（下：A4）
       - **设计 / 制图 / 审核**｜均填"老吴ai学习"
     - 符号格：第一角投影符号 + "一角法"（图形在上、字样在下，同置于符号格下部）
   - **标题栏不包含**：日期、版本、图纸张数；**不设版本修改记录表**。
5. **线宽**（ISO）：外框 0.5 / 轮廓 0.35 / 细线 0.25。
6. 模板文件：`图框模板_V4.dwg`（工作区），新图直接套用；**坐标级实现明细见附录 A**。
7. **视图缩放（V4.4 起）**：外图框绘制完成后，立即执行缩放至范围（ZOOM Extents，
   脚本写法 `doc.SendCommand("_.ZOOM _E ")`），将整张 A4 图框放大至当前 CAD 窗口
   适合的最大尺寸，便于实时观看与后续绘图检查。
8. **图线用途**：细虚线仅用于不可见轮廓；细点画线用于轴线/对称中心线，首末两端为长画；
   **中心对称线超出零件轮廓以 3~5 mm 为限，不得超出过多**（V1.11 强化）；
   细双点画线用于相邻辅助零件轮廓/可动件极限位置/中断线；粗点画线用于限定局部热处理或形位公差被测范围；
   粗虚线可表示允许表面处理的表面；断裂边界波浪线与双折线同图不混用。（GB/T 4457.4-2002）
9. **比例系列**：优先 1:1、2:1、5:1、1:2、1:5、1:10；允许 1:1.5、1:2.5、1:3、1:4、1:6、2.5:1、4:1
   （现用 1:3 属允许系列）；禁用系列外比例；与标题栏比例不同的视图/剖视，在该视图名称下方另行标注比例。（GB/T 14690-1993）
10. **字高系列**：1.8/2.5/3.5/5/7/10/14/20；汉字最小不低于 3.5 mm；现用 2.2/3.0/4.0 字高及宋体属企业偏离项，
    立档注明保留。（GB/T 14691-1993）
11. 幅面扩展预案：需要大于 A4 时按 A3→A2 顺序选用基本幅面；不留装订边格式边距 e：A4/A3/A2=10、A1/A0=20 mm；
    现 A4 内框 3 mm 属企业简化，立档注明。（GB/T 14689-2008）（备查）
12. 标题栏采用企业简化格式（GB/T 10609.1 裁剪版），立档声明；对外供图如客户要求国标完整标题栏，
    另行切换模板。（GB/T 10609.1-2008）（备查）

## 二、图号编码规则

- 格式：`项目代码-序号-机构代号+零件序号`，例如 `FP-001-A01`。
- 末尾字母 = 机构/功能组（如 A=治具功能件、B=升降/上料、C=吸盘/模组、D=旋转组件……），数字 = 零件序号，编号顺序对应装配层次。
- **镜像件 / 成对件各占一个图号、各出一张图**，不共用图纸。

## 三、视图表达习惯

1. 固定组合：**主视图 + 1~2 个侧视/端视图**（简单件至少主视+侧视）；**不画 3D 轴测示意图**。
2. 少用剖视；盘类、薄壁台阶件加局部全剖。**剖视标注用字母制"A-A"**：剖切位置用粗短画+箭头+大写字母，
   剖视图上方标"A-A"；单一剖切平面过机件对称面、且剖视图按投影关系配置、中间无图形隔开时可省略标注
   （GB/T 4458.6-2002 第6章）。**"剖面图"为已废止旧称，不得使用**；罗马数字仅用于局部放大图。
   **剖视比例可与主图不同**（如主图 1:2、剖视 1:1），需单独标注剖视比例。
3. 长轴类用断裂画法 + "对称"注法；大圆盘孔位用极坐标/角度标注。
4. 隐藏特征用虚线表达，背面沉孔在主视图用虚线圆表示。
5. 比例随件大小灵活选取，始终保持一张 A4：
   - 微小件放大：5:1、2:1
   - 中等件：1:1
   - 大件缩小：1:2 / 1:3 / 1:5 / 1:10
6. **比例评估前置（V4.3 起强制执行）**：开始画图前，先按零件实际尺寸与拟选比例估算视图纵向总高
   （主视图高 + 俯视/端视图高 + 视图间距 + 上下两侧尺寸标注占用），对照图框可用高度
   （内框高 204mm 扣除标题栏、技术要求区域后约 150mm）。**若布置后视图接近上下图框边缘
   （净距小于约 15mm），必须改用更小比例（1:2、1:3 等）重新估算、重新布置**，
   保证视图与上下图框边缘始终留有充足间距，不得触及图框；评估结论随交付报告一并说明。
7. **向视图**：不按投影关系配置的视图，上方标大写字母，相应视图旁用箭头指明投射方向并注相同字母；
   不写旧式"×向"。（GB/T 4458.1-2002）
8. 肋板/轮辐/紧固件/轴纵向剖切按不剖绘制（不画剖面线）；横向剖切照常；均布肋、均布孔不在剖切平面上时
   假想旋转到剖切平面后画出。（GB/T 4458.6-2002 §4.4/§5.4）
9. **局部放大图**：细实线圆圈出部位；多处放大用罗马数字编号，放大图上方注"Ⅰ 2:1"；仅一处时只注比例。（GB/T 4458.1-2002 §5.7）
10. **局部剖**：波浪线/双折线分界，波浪线不与任何图线重合、不超出轮廓、不穿过孔槽空缺；
    对称件半剖以细点画线为界，一半视图一半剖视。（GB/T 4458.6-2002 第5章）
11. **断面图**：移出断面轮廓用粗实线，尽量配置在剖切线延长线上，对称断面在延长线上时可省略标注；
    重合断面轮廓用细实线、不标注。（GB/T 4458.6-2002 第8章）
12. 局部视图断裂边界用波浪线/双折线，结构完整封闭时可省略；斜视图旋转配置时加旋转符号、字母靠近箭头侧，
    必要时注旋转角度。（GB/T 4458.1-2002 §5.1；GB/T 24739-2009）
13. 断裂画法细则：断裂边界用波浪线、双折线或细双点画线；断开后尺寸仍按实际长度标注。（GB/T 4458.1-2002 §5.5）
14. **重复要素**：规律分布的相同结构只画 1~2 个完整，其余用细点画线示位并标注总数；
    对称件视图可只画一半并加对称符号。（GB/T 16675.1-2012）
15. **投影法守则**：内部出图一律第一角、同一张图不得混用两种画法；接收第三角图样时先找标题栏投影符号、
    再按视图配置关系验证；转绘为第一角时左/右视图位置对调，严防镜像错误。（GB/T 14692-2008）
16. **剖面线**：金属件为 45° 等距细实线；同一零件在各剖视图中方向与间距一致；狭小剖面可涂黑。（GB/T 4457.5-2013）
17. 铸锻圆角处交线按过渡线画（细实线、端部与轮廓留间隙）；非圆相贯线允许用圆弧或直线简化。（GB/T 4458.1-2002 §5.3）（备查）
18. 网状/滚花在轮廓附近局部示意并在技术要求中注明；较小斜度/锥度结构由较大端画出、小端允许省略。（GB/T 16675.1-2012）（备查）

## 五、孔与螺纹标注句式（固定格式）

1. 通孔：`n×φd 完全贯穿`
2. 螺纹孔（双行式）：`φ底孔 ↓底孔深 / M×-6H ↓螺纹深`
   - 螺纹公差带一律 **6H**；底孔深度比螺纹深度大 2~4 mm（如 φ4.2↓12.4 / M5-6H↓10）。
3. 沉孔：`⌴φD ↓深度`，并注明"正面 / 反面"。
4. 标准孔型配方（沉头螺钉过孔，全批统一，便于刀具表管理）：
   - M3：φ3.4 + ⌴φ6.5↓3.4
   - M4：φ4.5 + ⌴φ8↓4.4
   - M5：φ5.5 + ⌴φ10↓5.4
   - M6：φ6.6 + ⌴φ11↓6.4
   - M8：φ9 + ⌴φ15↓8.6
   - 标注句式：M 系列通孔直接写螺纹规格 `n-M× 完全贯穿`（如 `6-M5 完全贯穿`），不写 `n-φd（M×）` 形式；
     沉头孔用双行句式 `n-φd 完全贯穿 / 沉孔φD 深h 正面（或反面）`，末尾不加括号螺纹规格备注。
5. 细牙螺纹（如 M8×1.0）仅用于限位 / 调节件；定位孔、精密小孔可在视图中涂黑强调。
6. 成组孔合并标注，一组指引线一次引出全部加工信息。
7. **沉孔符号语义**：⌴＝柱形沉孔或锪平（平底）；∨＝锥形沉孔（埋头孔），注锥角（如 ∨Φ13×90°）。
   现行"沉孔ØD 深h 正面"双行句式保留为企业统一句式，并明确其对应 90° 锥形沉头孔；锪平时注明"锪平"。（GB/T 16675.2-2012 表1）
8. **螺纹标记**：粗牙普通螺纹省略螺距（M5 即 M5×0.8）；细牙必须注螺距（M8×1.0）；左旋加注 LH；
   默认中等旋合长度 N 不注；内螺纹公差带大写（6H）、外螺纹小写。（GB/T 197-2003、GB/T 4459.1-1995）

## 六、尺寸标注习惯（双轨制）

> **尺寸注法总则（V1.3 起）**：①尺寸数值为机件真实大小的唯一依据，与图形大小及绘图准确度无关；
> ②默认 mm 不注单位；③所注尺寸为**最后完工尺寸**（含阳极氧化/镀层后状态），需镀前尺寸时必须注明"镀前"；
> ④同一尺寸一般只注一次，注在反映结构最清晰的图形上。（GB/T 4458.4-2003 第3章）

1. **大板 / 多孔板 → 坐标式标注**：
   - 对称件以零件中心为 0 原点（±值标注）；
   - 非对称件以板角（左下角或左上角）为 0 原点；
   - 直接列 X/Y 坐标值，防累积误差，CNC 编程友好。
2. **小件 → 链式 / 基准式标注**：总长总宽 + 分段尺寸。
3. 成组倒角 / 圆角集中标注：`n-C2`、`4-R5`。
4. 尺寸标注一律不得与图形干涉；直径尺寸优先斜向 45° 引出；视图间距适当放大；
   **视图及其全部标注组成的整体图形，必须布置在图纸可用空间的中间位置——
   整体包围盒中心与可用空间（内框以内、标题栏及左下角技术要求/未注公差表区域以上）
   中心对齐，容差 ±5mm**（V1.12 起，取代原"居中偏上"）。
5. **同侧平行尺寸：小尺寸在内（靠近视图）、大尺寸在外，由内向外从小到大依次布置（V1.2 起强制执行）。**
6. 对称机件只画一半或用断裂画法时：尺寸线略超过对称中心线或断裂边界，仅一端画箭头，注全尺寸值。（GB/T 4458.4-2003 §4.2.5）
7. 尺寸线、尺寸界线用细实线；尺寸线单独绘制，不得借用/重合其他图线或其延长线；尺寸界线自轮廓线/轴线/中心线引出
   并超出尺寸线约 2 mm；终端统一实心箭头、一图一种形式；同方向尺寸线间距均匀（≥5 mm）。（GB/T 4458.4-2003 §4.1/4.2）
8. 任何图线不得穿过尺寸数字，不可避免时将该处图线断开。（GB/T 4458.4-2003 §4.3）
9. 小尺寸无位置画箭头/注字时，允许用圆点或斜线代替箭头，数字引出标注。（GB/T 4458.4-2003 §4.2.6）
10. 角度注法：尺寸界线沿径向引出，尺寸线为以角顶点为圆心的圆弧，角度数字一律水平书写。（GB/T 4458.4-2003 §4.1.5/4.3）
11. 参考尺寸加圆括弧，不作加工检验依据。（GB/T 4458.4-2003 §5.4）
12. 板厚可在数字前加 t（如 t5）。（GB/T 4458.4-2003 §5.6）
13. 球面在 Φ/R 前加 S（SΦ/SR），不致误解时可省。（GB/T 4458.4-2003 §5.2）
14. 弧长数字左方加注"⌒"；弦长尺寸界线平行于弦的垂直平分线。（GB/T 4458.4-2003 §4.1.5/5.3）
15. 正方形断面边长前加"□"或"B×B"；由其他尺寸确定的半径只画尺寸线和 R、不注数字。（GB/T 4458.4-2003 §5.5/5.7）
16. **腰槽（长圆槽）竖向位置标注（V1.9 起，V1.10 修订）**：以槽两端半圆中心线之一为定位基准（任选其一），
    标注板边至该半圆心的距离；**不标注槽的中间中心**。**半圆中心线保持原长、不得延长**；
    尺寸界线到不了中心线时，**用细实线（DIM 层）自半圆中心线端部接画至尺寸界线**，
    即"引出基线加长"，而非拉长中心线。
17. **非两边对称的零件不画对称中心线（V1.10 起）**：零件整体无对称轴时，不画长对称中心线，
    仅保留孔、槽等局部特征的中心标记（短十字/短画）；仍有对称轴的方向（如仅左右对称）保留该向中心线。
18. **非两边对称方向的特征位置尺寸标注到零件外边缘（V1.11 起）**：零件在某方向不对称时，
    该方向上的特征（孔、槽等）位置尺寸必须以零件外边缘为基准标注，
    不得只注特征间距——例如竖向孔除标注孔距 58 外，还须标注零件下边缘到孔的距离 6；
    同侧多个尺寸仍按"小尺寸在内、大尺寸在外"布置。

## 七、公差分级体系

1. 一般尺寸 → 标题栏未注公差分段表（±0.05 ~ ±0.50）。
2. 一般孔位 → 未注线性尺寸公差表兜底（孔位精度要求更高时须单独标注）。
3. 定位销孔 → +0.02/0 或 +0.03/+0.01 单向公差（H7 级铰孔），必要时加注"仅限销孔"。
4. 关键孔距 → ±0.02；精密小件全尺寸 ±0.02；精密定位轴可到 ±0.01。
5. 形位公差只在关键功能件标注：
   - 铝板平面度 / 平行度 0.2（基准 A）；
   - 电木转盘收紧到 0.1；
   - 轴类圆跳动 0.05、圆柱度 0.03；配合外圆 Ra1.6。
6. 缓冲 / 挡块类厚度用单向负公差（如 -0.02/-0.05）。
7. **极限偏差书写**：上下偏差小数点对齐、小数点后位数相同（可用 0 补齐）；单向偏差为 0 时标 0 并与另一偏差个位对齐；
   对称偏差只注一次、前置 ±、与基本尺寸同字高；偏差字号比基本尺寸小一号；公差带代号与极限偏差两种形式全图统一选用，
   并注时偏差加圆括号。（GB/T 4458.5-2003 4.1/4.2）
8. 装配图配合代号用分数形式（分子孔、分母轴）；相配偏差孔上轴下；与标准件配合仅注自制件公差带。（GB/T 4458.5-2003 第5章）（备查）
9. 未注公差表下方加注"参照 GB/T 1804-f"；未注角度公差按 ±0.5°（或 GB/T 1804-f 分段）；
   未注倒角/倒圆极限偏差 ±0.2（0.5~3 段）；未注形位公差按 GB/T 1184-K 兜底。（GB/T 1804-2000、GB/T 1184-1996）
10. **粗糙度注法**：一律用去除材料符号（基本符号加短横）；符号尖端从材料外指向被测表面，
    注在可见轮廓线、尺寸界线或其延长线上；数字方向与尺寸数字一致；每一表面只注一次，重复要素（孔、槽）只注一处；
    右上角"其余"统一注法字高为图面其他粗糙度字号的 1.4 倍；全部表面同一要求时不注"其余"。（GB/T 131-2006）
11. **形位公差框格**：两格/多格，自左至右为特征符号｜公差值（圆/圆柱公差带前加 Φ、球形加 SΦ）｜基准字母；
    指引线细实线、弯折≤2 次；箭头指轮廓要素时与尺寸线明显错开、指中心要素（轴线/中心面）时与该要素尺寸线对齐；
    基准符号＝方框+涂黑或空心三角形以细线相连，字母大写水平书写，**禁用 E、F、I、J、L、M、O、P、R**；
    框格内字高与尺寸数字等高。（GB/T 1182-2018、GB/T 17851）

## 八、倒角 / 圆角分级

- 未注 C0.5 打底；
- 按件大小与功能单独引出：C1 / C2 / C3 / C5 / C10；精密小件 C0.2；
- 大底板棱边用圆角代替：R5 / R10 / R20 / R30；L 形座内转角 R15。
- 倒角 C 仅限 45°（C2=2×45°）；非 45° 倒角分别注角度与宽度（如 30°×1.5），不得写 C 形式。（GB/T 4458.4-2003 §5.9）

## 九、特殊注法与出图习惯

1. 成组加工要求用图面大字注记（如"4根等高处理"）。
2. 刻印要求注深度（如"数字+箭头，深 1mm"）。
3. **均布孔一律注 EQS**（GB/T 16675.2 正式缩写，义为"均布"）；**不使用 PCD 缩写**；
   分布圆直径作为普通尺寸注出（如 Φ80）；需要时附角度。
   **EQS 与成组孔加工信息合并为一组标注、EQS 置于最前**（如"EQS 6-M5 完全贯穿"）。
4. **图纸改版直接更新图面内容，不设版本修改记录表、不注版本号与日期**（V3 起）。
5. 出图前清理模板残留标注（如"⌴φ0 ↓0"之类无效引注）。
6. **引注位置规则（V1.4 起，V1.6/V1.7 强化）**：成组孔、沉头孔等引注文字距图形轮廓不宜过近（避免压迫图形，必要时向外/向上偏移）；倒角引注应**紧邻**被注倒角布置——引线取短（一般不超过约 10mm），引注文字位于被注倒角外侧就近的空旷处，不得远离倒角跨区布置（V1.7）；各引注之间保持充足间距，引注与尺寸线之间不得相互干涉。
7. **引注引线规则（V1.5 起，V1.6 修订）**：孔、倒角等引注的引线从图形引出时，靠图形一端必须带箭头，箭头大小与尺寸标注箭头一致（随 DIMASZ 设置）；**引线应为单段直线、不得折弯**。

## 十、执行总则

- **制作方式与交付格式（V4.6 起，V1.4 修订）：所有工程图一律在 AutoCAD 软件中实时制作（COM 自动化驱动 AutoCAD 绘图，
  Visible=True），绘图过程逐步可见——分步/分组绘制并插入短暂延时，且**每个实体创建后必须调用 Update()、
  每个阶段结束执行 Regen 强制画面刷新**（仅插入延时不触发重绘，否则屏幕只在最后一次性更新），
  使操作者能实时看到绘制过程；**脚本不改变 CAD 窗口状态（不置前、不最大化窗口）；
  图框绘出后即执行 ZOOM Extents，使整张 A4 图框充满当前 CAD 窗口，绘制全程保持该视野便于观看**；
  图纸制作完成后脚本不执行保存操作（禁止 SaveAs），由用户手动保存；最终保存格式为 DWG；
  不以 ezdxf/DXF 作为图纸交付格式（DXF 仅允许作中间校验用途；校验用 PNG 导出属打印操作，允许）。**
- 每张新图：先套用本规范第一章图框（模板 `图框模板_V4.dwg`）→ 第一角投影布置视图 → 按第五、六章标注 → 按第七章分级给公差 → 按第八章节倒角。
- 宁多一张图，不挤一张图；保持"一个零件一张 A4"。
- 规范未覆盖的特殊结构，参照最接近的标准句式引申，保持风格一致。

## 十一、效率措施与模板化（V1.3 起）

1. **规范先行**：新图动手前确认规范条款齐套并冻结，避免出图后改版返工。
2. **图框模板化**：图框+标题栏固化进模板文件（`图框模板_V4.dwg` 或 .dwt），新图以模板新建，不逐体重画图框。
3. **公共模块 `acad_common.py` 沉淀**：call() RPC_REJECTED 重试封装、TrueColor 颜色设置、横向 PlotToFile PNG 导出、
   实体审计（参照 `audit_fp007.py`）。
4. **固定实现方案（防重复试错）**：文字样式宋体；直径符号用 Φ(U+03A6)；线型 CENTER2/DASHED2 + LTSCALE 0.4；PNG 导出横向介质；引注引线＝单段直线 + AddSolid 实心三角箭头（不用 AddLeader——无模板新文档中 DIMLDRBLK/ArrowheadBlock 为空，箭头不显示）。
5. **干涉前置计算**：出图前先计算文字包围盒与尺寸线/视图间距；同侧尺寸小内大外由脚本自动排序位置，算好再画。
6. **AutoCAD 会话常驻复用**，减少启动开销。

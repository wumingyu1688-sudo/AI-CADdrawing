# AI制图规范V1.5（cad-drawing 技能包）

在 **AutoCAD** 中按《CAD制图的AI规范》**实时逐步绘制**机械工程图的 AI 技能包
（COM / pywin32 驱动，DWG 格式，逐步可见、自动校验、PNG 举证）。

## 内容清单

| 路径 | 说明 |
|---|---|
| `SKILL.md` | 技能入口：铁律、9 步流程、标注规则速查 |
| `references/CAD制图的AI规范V1.1.md` | **制图规范全文（当前版本 V1.12）**——图幅图框、视图表达、尺寸标注、公差分级、材料/表面处理分级、附录 A 图框坐标明细 |
| `references/制图规范存档说明.md` | 规范版本沿革与存档说明 |
| `references/pitfalls.md` | COM 自动化坑点清单（引线箭头、线型加载、RPC 拒绝等实证记录） |
| `scripts/acad_common.py` | 公共库：连接、环境、图层、文字、尺寸、引线、居中、自审计、PNG 导出 |
| `scripts/draw_template.py` | 绘图脚本模板（复制后按零件改几何与标注） |
| `scripts/smoke_test.py` | 公共库冒烟测试 |

## 使用前提

- Windows + 已打开的 AutoCAD（2019 及以上验证通过）
- Python 3.10+ 且安装 `pywin32`
- 图纸**全程不自动保存**：画完保持打开，人工 Ctrl+S 留存（技能铁律，防误写磁盘 .dwg）

## 规范要点（V1.12 摘录）

- A4 图框 + 第一角投影，标题栏含公司栏/材料/表面处理/数量/一角法符号
- 同侧平行尺寸**小内大外**；非对称方向特征位置尺寸**注到零件外边缘**
- 均布孔句式 `EQS 6-M5 完全贯穿`；沉头孔双行句式 `4-Φ6.6 完全贯穿 / 沉孔Φ11 深6.4 正面`
- 引线 = 单段直线 + 实心三角箭头，不得折弯；倒角引注紧邻被注倒角
- 中心线超出轮廓 3~5mm；非两边对称零件不画长对称中心线
- 视图整体（含标注）居于图纸可用空间中间（`acad_common.center_views`）

## 安装

将整个文件夹复制到 Agent 运行时的 skills 目录（如 `…/daimon/skills/AI制图规范V1.5/`），
技能索引刷新后即可通过"制图/绘图/工程图"等指令触发。
